能打开什么

几乎，什么都能打开

很多人以为阅读器只看文本。Omon 不是——图片、音频、视频、整套编程文件，它都直接打开。下列每个格式都在 app 里逐个实测过。

📄 文档document
Markdown · 纯文本(.txt) · HTML · Word(.docx) · PDF · PowerPoint(.pptx) · Excel(.xlsx / .xls)
📖 电子书e-book
EPUB　（封面 / 插图 / 多章的真实书籍，舒服的衬线阅读排版）
🖼 图像image
PNG · JPG / JPEG · GIF · WebP · SVG · BMP · ICO · TIFF / TIF
🎬 视频video
MP4 · MOV · M4V · WebM　（app 内窗口播放）
🎵 音频audio
MP3 · WAV · M4A · AAC · FLAC · OGG
💻 代码 & 数据code
JS / TS · Python · Rust · Go · Java · C / C++ · Swift · SQL · Shell · Vue / Svelte · JSON / YAML / TOML / XML / CSV · Dockerfile / Makefile…
几十种编程语言与配置文件 + 任何纯文本文件，自动语法高亮。

视频 / 音频走系统播放器，具体编解码跟随你的 macOS 版本。

产品哲学
Observe More, Organize Naturally
看见更多，自然整理 —— 这是 Omon 名字的由来，也是它做事的方式：看懂更多格式，自然地整理与编辑。

安装（内测版）

  1. 点上方「下载」，拿到 Xiaoer-Omon-x.x.x.dmg。
  2. 打开 DMG，把 Xiaoer Omon 拖进「应用程序」。
  3. 首次打开：右键 App →「打开」→「打开」。若没有该选项，去系统设置 → 隐私与安全性点「仍要打开」；或终端跑 xattr -dr com.apple.quarantine "/Applications/Xiaoer Omon.app"
内测版尚未做苹果公证，所以首次需绕过一下。正式公证版下载即开，无需任何操作。
